
# LKPD Digital Interaktif

Sebuah platform pendukung pembelajaran yang memungkinkan Guru untuk mengelola kuis dan Siswa untuk mengerjakan soal secara interaktif dengan umpan balik langsung.

## Fitur Utama:
- **Role Guru**: Login tersembunyi, CRUD Soal, Monitoring Nilai Siswa, Dashboard Statistik.
- **Role Siswa**: Pemilihan kelas, input nama, pengerjaan kuis interaktif, review jawaban benar/salah, skor instan.
- **UI Modern**: Menggunakan skema warna biru profesional, font Space Grotesk dan Inter, serta layout responsif.

## Teknologi:
- Next.js 15 (App Router)
- Tailwind CSS
- Shadcn UI
- Lucide React Icons
- Recharts for Dashboard
