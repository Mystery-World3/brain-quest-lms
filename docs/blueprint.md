# **App Name**: LKPD Digital Interaktif

## Core Features:

- Teacher Authentication & Authorization: Secure login system for teachers, with access to a dedicated dashboard. This access point is subtly integrated, for example, by clicking on a logo on the landing page.
- Class & Quiz Management (Teacher): Teachers can create, read, update, and delete (CRUD) classes and quiz questions specific to each class and material. This includes defining questions, answer options, and the correct answer.
- Student Class & Name Selection: A scroll-down menu on the landing page allows students to select their class. After selecting a class, students input their name to start the quiz.
- Student Quiz Interface: Students can navigate through and answer quiz questions. After completing all questions, a 'Selesai' (Done) option is available at the bottom to submit their answers.
- Instant Quiz Feedback & Review: Upon submission, students receive immediate feedback showing their final score and indicating which answers were correct or incorrect, with an option to review.
- Teacher Performance Dashboard: Teachers can view student scores, quiz performance, and progress for each class through a dedicated dashboard interface.

## Style Guidelines:

- Color Scheme: Light theme, emphasizing clarity and focus, suitable for a learning environment.
- Primary Color: A calming and professional blue (#2666B9), conveying reliability and concentration.
- Background Color: A very light, desaturated blue (#ECF0F3), providing a soft, non-distracting canvas.
- Accent Color: A vibrant aqua-blue (#36CBF2), drawing attention to interactive elements and calls to action without being overwhelming.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, digital, and engaging feel in titles and key headings.
- Body Font: 'Inter' (sans-serif) for its high legibility and neutral yet modern appearance, ideal for questions, explanations, and other textual content.
- Utilize a consistent set of clean, simple, and intuitive icons related to education, quizzes, and user navigation, to enhance user experience.
- A clean, structured layout with clear content hierarchy. The landing page features a prominent, scrollable class selection, while dashboards prioritize data readability.
- Subtle and functional animations to provide visual feedback on user interactions, such as question submission or navigation transitions, enhancing the interactive feel.
